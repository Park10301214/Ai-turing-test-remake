# LLM Turing Test Analyzer

ChatGPT, Gemini, Claude와 실제 사람의 응답을 비교하기 위한 웹 기반 튜링 테스트 프로젝트입니다.

## 파일 구성
- `index.html`: 시작, 실험, 결과, 관리자 화면
- `style.css`: 반응형 디자인
- `questions.js`: 질문과 응답 데이터
- `script.js`: 실험 진행, 자동 채점, CSV, 통계 기능

## GitHub Pages 배포
1. 새 GitHub 저장소를 만듭니다.
2. 위 네 파일을 저장소 최상위에 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/root`로 선택하고 저장합니다.
6. 잠시 뒤 생성된 주소로 접속합니다.

## 실제 실험 전 필수 수정
`questions.js`의 예시 응답을 실제로 ChatGPT, Gemini, Claude, 사람에게서 받은 답변으로 교체하세요. 모든 AI에는 같은 질문과 같은 지시문을 사용해야 합니다.

권장 공통 지시문:
> 아래 질문에 한국어로 3~5문장으로 답해 주세요. 스스로를 AI라고 밝히거나 모델 이름을 언급하지 마세요. 지나치게 형식적인 제목이나 목록은 사용하지 마세요.

## 실험 절차
1. 참가자 10명이 각자 실험에 참여합니다.
2. 각 참가자는 8개의 익명 응답을 사람 또는 AI로 판단합니다.
3. 결과는 해당 기기의 브라우저 `localStorage`에 저장됩니다.
4. 같은 기기에서 관리자 통계를 열어 전체 결과를 확인합니다.
5. 전체 결과 CSV를 다운로드해 Google Sheets에서 표와 그래프로 정리합니다.

## 주의
GitHub Pages만으로는 서로 다른 참가자의 휴대폰 결과를 자동으로 한곳에 합칠 수 없습니다. 가장 간단한 방법은 한 대의 기기를 순서대로 사용하거나, 각 참가자가 개인 CSV를 내려받아 연구자에게 전달하도록 하는 것입니다.


## 8문항 구성
- 일상 대화 2문항
- 감정 표현 2문항
- 창의적 사고 2문항
- 논리 추론 2문항
- ChatGPT, Gemini, Claude, Human 응답은 각각 2개씩 출제됩니다.

참가자는 이름 또는 별명만 입력하며 참가자 번호는 입력하지 않습니다. 각 문항에서는 선정 이유를 미리 제시된 항목 중 하나로 선택할 수 있습니다.
